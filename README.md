# Suica Usage Enhancer (Chrome拡張)

モバイルSuicaの利用履歴ページ（`https://www.mobilesuica.com/iq/ir/SuicaDisp.aspx`）を以下のように強化します。

- 日付範囲をカレンダー（`type="date"`）で指定して絞り込み
- 種別チェックボックスで絞り込み（入／出／定期／物販など）
- 場所・駅名キーワードフィルター
- 休日入力（YYYY-MM-DD複数指定）＋週末で色分け
- 元のテーブルをワンクリックで隠す表示切替

## 使い方
1. `manifest.json` を読み込んで拡張機能をデベロッパーモードで追加（パッケージ化せずフォルダを指定）。
2. `https://www.mobilesuica.com/iq/ir/SuicaDisp.aspx` を開く。
3. ページ上部に「Suica利用履歴ビュー（拡張）」パネルが表示されるので、日付・種別・キーワード等を指定して「フィルター適用」をクリック。
4. 必要なら「元テーブルを隠す」をONでオリジナルの表を非表示。

## 実装メモ
- コンテンツスクリプト：`content.js`
- スタイル：`styles.css`
- マッチURL：`https://www.mobilesuica.com/iq/ir/SuicaDisp.aspx*`
- 祝日の色分けは手入力した日付と週末（土日）のみ。追加ロジックが必要ならAPI連携を検討してください。

## 既知の制限
- 年跨ぎ判定は「表示対象」の年を基準に、選択月より大きい月を前年扱いにしています。表示期間が更に遡る場合はロジック拡張が必要です。
- 印刷機能など、元ページのフォーム送信は変更していません（拡張テーブルは閲覧専用）。
